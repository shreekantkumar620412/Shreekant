# 📸 Raj Photography Website

## Project Info
- Framework: React (JSX)
- Styling: Inline CSS (no external libraries needed)
- Font: Google Fonts (Playfair Display + Inter)

## Features
- 🏠 Home Page with Hero, Gallery Preview, Packages, About
- 🖼️ Full Gallery with Category Filter
- 🔐 Login & Registration System
- 👑 Admin Panel (Owner Only)
- 💳 Payment Gateway (Card, UPI, Net Banking, COD)
- 💬 Live Chat Widget
- 🔒 Privacy Policy Page

## How to Run
1. Install Node.js
2. Create React app: `npx create-react-app photo-site`
3. Replace `src/App.js` with `photography-website.jsx`
4. Run: `npm start`

## OR use directly on Claude.ai
Paste the JSX file content into Claude Artifacts (React mode)

## Tech Stack
- React Hooks (useState, useEffect, useRef)
- CSS Variables
- Google Fonts API
- Unsplash (photo CDN)
